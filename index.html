Personal Projects

SiteStall

Mozilla browser extension written in JavaScript, HTML5 and CSS3. The intention behind this project is to promote productivity while using the web. Developed with the user in mind, the backend is where the magic happens with this app... SiteStall is capable of managing threshold intervals and blocking browser activity based on user feedback.

Image Processor

Program which offers 20+ different filters which can be used to enhance images formats to produce stunning results. Written in C++, this work-horse program manages memory efficiently to produce optimal image renderings based on pixel array density.


Hover over each image to see an example enhancements!



Siren

Fully featured social media web application for sharing music. Implemented with a React/JS front-end which utilizes Redux for state management. The front-end is connected to a Node.js/Express.js REST API and utilizes material-ui for component design. Data is stored within a NoSQL fire-store database, although I am in the process of transitioning this to PostgreSQL.
